# My Wallet

O My Wallet é uma carteira digital onde o usuário pode registrar seus ganhos e gastos de forma fácil e ágil.

Use agora neste link https://mywallet-frontend-five.vercel.app

## Sobre

Esta aplicação possui os seguintes recursos disponíveis para o usuário:

- Registro de usuário
- Login
- Lista de todos os registros do usuário
- Adicionar um registro de entrada
- Adicionar um registro de saída
